How it works

Three steps. One command.

1

You name a topic

Any subject: kubernetes networking, GDPR compliance, machine learning algorithms. Plain English.

2

Claude builds the graph

Auto-generates an ontology, discovers authoritative sources, crawls, chunks, and parses with GPT-4o-mini.

3

Query your own database

A structured knowledge graph in Apache AGE (PostgreSQL). No vendor lock-in. No hosting fees. Just SQL + Cypher.
